<h1>How To</h1>

<h3>Init</h3>
<ul>
    <li>
        npm install yarn -g
    </li>
    <li>
        npm i -g babel-cli
    </li>
    <li>
        npm i -g webpack-cli
    </li>
    <li>
        npm install / yarn install
    </li>
</ul>

<h3>Start</h3>

<ul>
    <li>
        $ yarn build / yarn webpack
    </li>
    <li>
        $ yarn dev
    </li>
</ul>
